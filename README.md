# dormitory-system# 智能宿舍管理系统

## 项目介绍
基于 SpringBoot + Vue3 的前后端分离项目，实现宿舍管理、学生管理、多角色权限控制等功能。

## 技术栈
SpringBoot / Vue3 / MySQL / Axios

## 功能模块
- 多角色登录（学生 / 宿管 / 管理员）
- 学生信息管理（CRUD）
- 请假申请 / 审核流程
- 数据展示

## 项目运行

### 后端
cd backend
mvn spring-boot:run

### 前端
cd frontend
npm install
npm run dev

## 项目说明
用于学习Web开发流程，包含完整前后端交互及数据库操作
